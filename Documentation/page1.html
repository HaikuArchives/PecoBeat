The Programs

The PecoBeat package consists of two programs:
  • PecoBeat - the main program   and
  • InternalMIDI - the connection to the BeOS MIDI synthesizer

InternalMIDI

InternalMIDI works as a virtual device. In a future version PecoBeat will support more devices, but at the moment it only supports the internal sequencer via InternalMIDI.

The InternalMIDI icon in the Deskbar
When you start PecoBeat, InternalMIDI is started automatically. You can see a window which tells you that the instruments are loaded. Then InternalMIDI disappears to the Deskbar where you can see it as a little icon.

Just double-click on the InternalMIDI icon to get the settings window! The settings window should be very self-explaining.

If you need help on InternalMIDI, please read the documentation that comes with that program!
